üíª Token Endpoint Code

Simple implementation example

// Token Endpoint Approach - Anonymous Access import ReactWebChat, { createDirectLine } from 'botframework-webchat'; const TOKEN_ENDPOINT = 'https://<env>.api.powerplatform.com/.../directline/token'; async function initChat() { // Step 1: Fetch token (no auth required) const response = await fetch(TOKEN_ENDPOINT); const { token } = await response.json(); // Step 2: Create DirectLine connection const directLine = createDirectLine({ token }); // Step 3: Render chat return <ReactWebChat directLine={directLine} />; }

üéØ Decision Guide

Which approach should you choose?

üé´ Use Token Endpoint

  • Public-facing website
  • Users should NOT sign in
  • Public knowledge sources only
  • Simpler deployment preferred
  • Citizen services / customer support

üîë Use M365 Agents SDK

  • Enterprise internal app
  • Bot needs user identity
  • SharePoint/Dataverse access
  • Teams or SharePoint integration
  • SSO with Microsoft 365
